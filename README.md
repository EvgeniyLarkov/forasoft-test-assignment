# ForaSoft test assignment
### demo: https://mysocketschat.netlify.app/
## Stack

- Typescript
- React
- Socket.io
- fastify

## Installation

```
$ npm ci
```

## Running-server

```
$ npm run-script start / make start-server
```

## Running-client

```
$ npm run-script develop / make develop
```

## Build

```
$ npm run build / make build
```
